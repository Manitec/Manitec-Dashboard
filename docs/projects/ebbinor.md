# Ebbinor

Ebbinor is a voxel game project built on the Minetest open-source engine, written in C++.

## Status
- **Status:** Active
- **Language:** C++
- **Engine:** Minetest (voxel game engine)

## Tools Used
| Tool | Purpose |

| [Cloudflare](../infra/tools.md) | DNS & CDN |
| [Namecheap](../infra/tools.md) | Domain management |
| [NightCafe](../infra/tools.md) | AI art / texture generation |

## Notes
Built on the Minetest voxel engine (LGPL v2.1+). Supports easy modding,
game creation, Docker deployment, and cross-platform builds (Linux, Windows, macOS).
